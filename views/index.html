Installation

Get Started in Minutes

Choose the installation method that fits your workflow.

Add the script tag to your page:

<script src="https://cdn.jsdelivr.net/npm/hugerte@1/hugerte.min.js"></script>

Then initialize the editor:

<textarea id="editor"></textarea>
<script>
  hugerte.init({
    selector: '#editor',
    plugins: 'link lists image table code',
  });
</script>
npm install hugerte

Then import and initialize in your JavaScript:

import hugerte from 'hugerte';
import 'hugerte/models/dom';
import 'hugerte/themes/silver';
import 'hugerte/icons/default';
import 'hugerte/skins/ui/oxide/skin.js';
import 'hugerte/skins/ui/oxide/content.js';
import 'hugerte/skins/content/default/content.js';

hugerte.init({
  selector: '#editor',
  skin_url: 'default',
  content_css: 'default',
});

See the bundling docs for the full list of imports (plugins, skins, etc.).

yarn add hugerte

Then follow the same import pattern as the npm tab above.

composer require hugerte/hugerte

You can also download the latest release as a zip from the hugerte-dist tags page.

Need more? See the full documentation for bundling, localization, plugins, skins, and advanced configuration.
